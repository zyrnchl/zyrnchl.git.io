About the Project

A fully functional furniture e-commerce website designed and developed from scratch. In collaboration with Snckish, the platform allows customers to browse a catalog of furniture products, add items to their cart, and complete purchases through a streamlined checkout process.

Built with a responsive design using Bootstrap 5, the site works seamlessly across all devices. The backend is powered by PHP and MySQL, enabling dynamic product management and secure order processing.

Project Screenshot

Key Features

Shopping Cart & Checkout

Users can add items to cart, update quantities, and complete orders through a smooth checkout flow.

Admin Panel

A dedicated admin dashboard to manage products, view orders, update inventory, and track sales.

Inventory Tracking

Real-time stock management — automatically updates product availability based on orders placed.

Responsive Design

Fully optimized for desktop, tablet, and mobile using Bootstrap 5 grid and components.
