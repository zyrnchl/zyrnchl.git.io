About the Project

Warm, Polished Visual Style

This project creates a welcoming online furniture shopping experience by using soft earth tones, rounded panels, and a modern design system that makes browsing feel professional yet inviting.

Product-First Layout

The design emphasizes furniture items as the centerpiece of the site. Large product cards, clean typography, and spacious imagery guide users smoothly from browsing to checkout.

Collaborative Project

Developed with my classmate Snckish, this project demonstrates teamwork in both design and backend development, resulting in a polished, user‑friendly e‑commerce platform.

Responsive Design System

Built with Bootstrap 5, the interface adapts seamlessly across desktop, tablet, and mobile, ensuring customers can shop conveniently from any device.

Key Features

Shopping Cart & Checkout

This feature allows customers to add products to their cart, adjust quantities, and complete purchases through a streamlined checkout process. It ensures a smooth shopping experience from browsing to payment.

Admin Panel

The admin dashboard provides store managers with full control over the platform. They can manage product listings, monitor customer orders, update inventory, and track overall sales performance in real time.

Inventory Tracking

Stock levels are automatically updated whenever an order is placed. This prevents overselling, keeps product availability accurate, and ensures customers always see the latest inventory status.

Responsive Design

The site is fully optimized for desktop, tablet, and mobile devices using Bootstrap 5. This guarantees that customers can shop comfortably no matter what device they are using.
